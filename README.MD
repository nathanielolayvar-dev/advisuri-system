## ðŸš€ Getting Started with Docker (Recommended)

This is the fastest path to deployment. No need to install Python, Node, or PostgreSQL locally.

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd advisuri-system
```

### 2. Launch the stack
```bash

```

### 3. Access the services
| Service      | URL                       |
|-------------|---------------------------|
| Frontend    | http://localhost:5173     |
| Backend API | http://localhost:8000     |
| Database    | PostgreSQL on port 5432   |


## ðŸ›  Manual Setup (Local Development)
If you prefer to run services natively in separate terminals:

### Backend (Django)
```bash
cd backend
python -m venv venv
```
### Activate virtual environment

- Windows:
```bash
.\venv\Scripts\activate
```
- macOS / Linux:
```bash
source venv/bin/activate
```
- Install dependencies and run server
```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend (React + TypeScript)
```bash
cd frontend
npm install
npm run dev
```
